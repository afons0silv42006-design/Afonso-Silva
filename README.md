# Tema do Projeto

- Escolher um **tema** ou **idéia** a propor ao professor
- Colocar no **ReadMe** a **Documentação** do projeto.
- Linguagem deve ser o **Bash**, não o _Phyton_


O meu Projeto vai ser um script que que gera um pequeno relatório sobre o estado do seu computador, mostrando o uso de CPU, memória RAM e espaço em disco. 

O objetivo deste projeto é conseguir facilitar a visualização rápida do consumo de hardware e fazer um relatorio com informações críticas do sistema, com as seguintes informações:

**Uso do CPU**
**Memória RAM**
**Disco Rígido**
**Alerta em vermelho caso algum recurso ultrupasse os 80**
